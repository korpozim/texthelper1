# texthelper1

### Install texthelper1

```
npm install --save texthelper1
```

### Using texthelper1

```javascript
const helper = require("texthelper1");

let text = "asd";

console.log(helper.endpoint(text));  // asd.
console.log(helper.firstup(text));   // Asd
```
